# Degraded Mode

Implements: R-6.10.7 [verified], R-6.10.9 [verified]

## Overview

The system degrades by isolating the smallest safe scope: command, drive,
observer, item, or scope block. It does not silently keep going when the
current source of truth is missing or untrusted.

Stable degraded-mode IDs make the matrix auditable. Each row below maps to at
least one exact test name so the documentation can point at executable
behavior instead of broad package-level claims.
